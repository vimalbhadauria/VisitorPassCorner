# VisitorPassCorner

VisitorPassCorner is a Visitor Management System developed using JSP, Servlet, JDBC, and MySQL.

## Features
- User Login & Registration
- Add Visitor
- Visitor Dashboard
- Session Handling
- Filter Validation
- Logout Functionality

## Technologies Used
- Java
- JSP
- Servlet
- JDBC
- MySQL
- HTML
- CSS
- Bootstrap

## Project Structure
- Controller
- DAO
- Model
- Filter
- JSP Pages

## Tools Used
- Eclipse IDE
- Apache Tomcat Server
- Git & GitHub
